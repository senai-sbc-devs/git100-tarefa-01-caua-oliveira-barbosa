este é um título

o objetivo da atividade é trabalhar em dupla usando o git

Clone

'git clone -url_da_branch-' - responsável por clonar um repositório para o ambiente local

Branch

'git branch-nome_da_branch-' - responsável por criar uma branch localmente.

Commit

'git commit -m "mensagem"' - responsável por fazer o commit das alterações nos arquivos adicionando uma mensagem.

Merge

'git merge -nome_da_branch-' - faz a junção de uma branch com outra.

Pull request

O pull request é um pedido feito para o dono do repósitório para juntar o código feito com uma das branchs do repo.
